Welcome to my unusual DIY CNC milling machine project.  The video series is available to follow along on my YouTube channel under my channel name "routerCNC".
The milling machine is my own design and has no quill but instead uses an all moving head which slides on the main column and is controlled by an AC servo driving a ballscrew.
The control system will be based on the popular ESP32 microprocessor which will take user inputs via the control boxes and move the head accordingly.
This means it needs lots of lovely code to control it all!
At the moment this I have some early code to control the basic functions but I expect it will develop over time and end up somewhere quite different.
If you enjoy reading other peoples strange code then feel free to dip into each of the code folders; these will increment with version number so you can look at the progress.
Thanks for reading and following along.
