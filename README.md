Welcome to my unusual DIY CNC milling machine project.  The video series is available to follow along on my YouTube channel under my channel name "routerCNC".
Link is here: https://www.youtube.com/watch?v=ioo_Vyz_lmk
The mechanical parts were CNC machined by JLCCNC. It is my own design and features a fully moving head which means the quill function is fully electronic and configurable.
This repository will contain various iterations of the ESP32 code for anyone to follow along.

The milling machine has no quill but instead uses an all moving head which slides on the main column and is controlled by an AC servo driving a ballscrew.
The control system will be based on the popular ESP32 microprocessor which will take user inputs via the control boxes and move the head accordingly.
This means it needs lots of lovely code to control it all!

At the moment I have some early code to control the basic functions but I expect it will develop over time and end up somewhere quite different.
I've started at V25 of the code as the previous versions were just building up the functionality.  It is subject to lots of change as the project progresses.
If you enjoy reading other peoples strange code then feel free to dip into each of the code folders as they evolve; these will increment with version number so you can look at the progress.
Thanks for reading and following along.
